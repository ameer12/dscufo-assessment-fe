# FrontEndDSCUFO (ECHO / dscufo.xyz)

Marketing site frontend: homepage, intelligence cards, and echo pages.

## Excluded from this export

- `node_modules`, `dist`, nested `FrontEndErgonia`
- Career pages, hiring intake, admin console, verification flow
- Local env files (add your own `.env` if needed)

## Setup

```bash
npm install
npm run dev
npm run build
```

## Routes

- `/` — Home
- `/echo-world`, `/echo` — Network pages
